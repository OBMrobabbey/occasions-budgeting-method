# Occasion Budgeting Method (OBM)

Created by Rob, 2026.

The Occasion Budgeting Method (OBM) is an annualised behavioural framework that uses a fixed allocation of “Occasions” to regulate alcohol consumption through pre‑commitment, scarcity, and behavioural budgeting. OBM defines Occasions as pre‑declared periods (from single events to multi‑day holidays) during which alcohol use is permitted under a fixed annual allocation.

Full documentation is held privately by the author. This repository exists to establish authorship and timestamp of the method and its core terminology.
